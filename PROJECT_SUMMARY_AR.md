## 🏗️ البنية المعمارية

### قاعدة البيانات
- **النوع:** PostgreSQL
- **الموفر:** Supabase (مجاني)
- **ORM:** Drizzle ORM
- **الجداول:** 11 جدول

### الخلفية (Backend)
- **Framework:** Next.js 16 (App Router)
- **اللغة:** TypeScript
- **المصادقة:** JWT مع bcrypt
- **الحماية:** Middleware + Role-based access

### الواجهة (Frontend)
- **Framework:** React 19
- **التصميم:** Tailwind CSS 4
- **الرسوم البيانية:** Recharts
- **الأيقونات:** Lucide React
- **PDF:** jsPDF

### النشر (Deployment)
- **الاستضافة:** Netlify (مجاني)
- **قاعدة البيانات:** Supabase (مجاني)
- **CI/CD:** GitHub + Netlify Auto-deploy

## 👥 الأدوار والصلاحيات

### 1. Super Admin (المدير الأعلى)
- **الحد الأقصى:** 3 حسابات
- **الصلاحيات:**
  - إدارة جميع الشركات (إنشاء، تفعيل، إيقاف، حذف)
  - اعتماد حسابات المحاسبين
  - إدارة خطط الاشتراك
  - قفل الحسابات المنتهية
  - عرض جميع الإحصائيات
  - سجل التدقيق الكامل

### 2. Admin (مدير الشركة)
- **النطاق:** شركته فقط
- **الصلاحيات:**
  - إدارة المشتركين (CRUD)
  - إنشاء خطط الدفع
  - إدارة الديون والفواتير
  - عرض التقارير
  - إعدادات الشركة
  - رفع شعار الشركة
  - تخصيص الإيصالات

### 3. Accountant (المحاسب)
- **الشرط:** موافقة Super Admin
- **الصلاحيات:**
  - إضافة المدفوعات
  - طباعة الإيصالات
  - تصدير PDF
  - عرض المشتركين
  - البحث عن العملاء
  - **لا يمكن:** تغيير إعدادات الشركة

## 📊 المميزات المنفذة

### ✅ لوحة التحكم
- 5 بطاقات إحصائية (إجمالي المشتركين، النشطين، المنتهيين، الديون، الإيرادات)
- رسم بياني للإيرادات الشهرية (6 أشهر)
- رسم دائري لتوزيع الاشتراكات
- آخر المدفوعات
- الاشتراكات المنتهية قريباً

### ✅ إدارة المشتركين
- إضافة/تعديل/حذف المشتركين
- البحث بالاسم أو الهاتف
- التصفية حسب الحالة
- عرض تفاصيل المشترك
- سجل المدفوعات والديون
- دعم 4 أنواع اشتراك (شهري، ربع سنوي، نصف سنوي، سنوي)

### ✅ المدفوعات والإيصالات
- تحصيل المدفوعات (نقداً، بطاقة، تحويل، شيك)
- إصدار إيصالات احترافية
- تصدير PDF مع:
  - شعار الشركة
  - معلومات الشركة
  - رقم الإيصال
  - المبلغ بالأرقام والحروف (عربي/إنجليزي)
  - تفاصيل العميل
- طباعة مباشرة
- دعم 4 طرق دفع
- دعم 4 أنواع مدفوعات (اشتراك، سداد دين، دفعة مقدمة، استرداد)

### ✅ إدارة الديون
- عرض المدينين
- إجمالي الديون
- سجل الديون لكل مشترك
- سداد جزئي/كلي

### ✅ التقارير (6 أنواع)
1. **التحصيل اليومي:** الإيرادات حسب اليوم مع رسم بياني
2. **التحصيل الشهري:** الإيرادات حسب الشهر
3. **تقرير المدينين:** قائمة بالمدينين ومبالغهم
4. **الاشتراكات المنتهية:** المشتركين الذين تنتهي اشتراكاتهم قريباً
5. **تقرير الإيرادات:** إجمالي الإيرادات مع رسوم بيانية
6. **أداء المحاسبين:** عدد المدفوعات والمبالغ لكل محاسب
- تصدير CSV لجميع التقارير

### ✅ الاشتراكات
- 5 خطط (تجريبي، شهري، ربع سنوي، نصف سنوي، سنوي)
- إدارة الصلاحية
- قفل تلقائي بعد الانتهاء
- وضع القراءة فقط بعد الانتهاء
- سجل تجديد الاشتراكات

### ✅ الإعدادات
- معلومات الشركة (الاسم، الهاتف، البريد، العنوان)
- تخصيص الإيصال (اللون، الترويسة، الفوتر)
- تغيير اللغة (عربي/إنجليزي)
- تغيير المظهر (داكن/فاتح/تلقائي)
- عرض حالة الاشتراك

### ✅ Super Admin Panel
- إحصائيات عالمية (إجمالي الشركات، المشتركين، الإيرادات)
- إنشاء شركات جديدة
- تفعيل/إيقاف الشركات
- تجديد الاشتراكات
- اعتماد المحاسبين
- قفل الحسابات المنتهية
- سجل التدقيق الكامل

## 🔐 الأمان

### حماية المسارات
- Middleware يتحقق من الجلسة
- إعادة التوجيه إلى صفحة تسجيل الدخول
- حماية مسارات Super Admin

### عزل المستأجرين
- كل استعلام يصفّي حسب `tenantId`
- لا يمكن الوصول لبيانات شركات أخرى
- Super Admin فقط يرى الكل

### المصادقة
- JWT مشفّر (HS256)
- bcrypt لكلمات المرور (12 rounds)
- HttpOnly cookies
- انتهاء الصلاحية بعد 7 أيام

### الصلاحيات
- التحقق من الدور في كل عملية
- المحاسبون مقيدون
- وضع القراءة فقط للشركات المنتهية

### سجل التدقيق
- تسجيل كل عملية (إنشاء، تعديل، حذف)
- حفظ: المستخدم، الإجراء، الكيان، الوقت
- Super Admin فقط يرى السجل
